## ğŸ› ï¸ Setup & Development

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/gavinbot367/gavinnot.github.io.git
   cd gavinnot.github.io
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the backend server** (optional, for chatbot API)
   ```bash
   npm start
   ```
   Server will run on http://localhost:3001

4. **Open in browser**
   Simply open `index.html` in your browser, or use a local server:
   ```bash
   python -m http.server 8000
   # or
   npx serve
   ```

### GitHub Pages Deployment

The site is automatically deployed via GitHub Pages. Any push to the `main` branch will update the live site at:
https://gavinnot.github.io/
